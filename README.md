### Informations
Frontend `React.js + Vite`
Backend `Express.js`
Database `MongoDB`

### Get Started
Get the `.env` from the team, then put it in `server/`.

Install dependencies
```bash
npm install
```

Start Server
```bash
cd server
npm start
```

Start Client
```bash
cd client
npm run dev
```
